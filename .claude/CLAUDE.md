# BlinkOn Presentation

This repo contains a presentation for BlinkOn about CSS Gap Decorations — a proposed CSS feature for styling gaps in flexbox, grid, and multi-column layouts. The presentation covers the feature's history, CSS Working Group feedback, implementation progress in Chromium, and future vision.

## Repo Structure

- `gap-decorations-v2.html` — the main presentation (HTML/CSS/JS, self-contained)
- `slide-content.md` — companion markdown with slide content and structure (source of truth for wording)
- `gap-decorations-presentation.html` — earlier version of the presentation
- `gap-decorations-timeline.html` — timeline visualization of the feature's development
- `gap-decorations-history.md` — detailed history of the gap decorations spec and implementation
- `feedback-tracker.md` — tracking CSS WG and community feedback on the proposal
- `f2f-january-2026.md` — notes from the January 2026 CSS WG face-to-face meeting
- `paint-scrolling-fixes.md` — notes on paint/scrolling implementation fixes in Chromium
- `vision-part-2.md` — future vision and next steps for gap decorations

## File Relationships

- `gap-decorations-v2.html` — the actual presentation (HTML/CSS/JS)
- `slide-content.md` — companion markdown with slide content, structure, and links

**When updating slide content or wording**, always update both files together. The markdown is the source of truth for content; the HTML is the rendered version.

**Styling-only changes** (CSS, layout, animations, diagrams) do not require updating `slide-content.md`.
